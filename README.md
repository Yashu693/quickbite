## 🌟 Overview

QuickBite is a highly-interactive, beautifully animated application engineered to give college students a seamless, premium food ordering experience. Featuring **Cinematic 3D Transitions**, **Glassmorphism UI elements**, and a **Mobile-First App-like Navigation**, QuickBite redefines local food delivery interactions.

## ✨ Features

- 🎭 **Cinematic Transitions:** Breathtaking navigation effects powered by Framer Motion, including shockwaves and emoji particle animations during "College Selection".
- 📱 **Mobile-First Layout:** A sleek app-like bottom navigation (`BottomNav`) ensures smooth usage on any device.
- 💳 **Seamless Flow:** Full integration of Cart, Payment, History, and Receipt views.
- 🚀 **Performant & Fast:** Built on Vite for lightning-quick HMR, compiled with standard JavaScript.
- 🔥 **Serverless Backend:** Tightly integrated with Firebase for realtime data tracking and responsive rules (`firestore.rules` & `storage.rules`).

## 🛠 Tech Stack

- **Framework:** React 19 + React Router DOM v7
- **Animations:** Framer Motion
- **Build Tool:** Vite
- **Database & Auth:** Firebase
- **Deployments:** Vercel

## 🚀 Getting Started

1. **Clone the repository:**
   ```bash
   git clone https://github.com/Yashu693/quickbite.git
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Configure Environment:**
   - Update `.env` / `.env.local` with your Firebase project credentials.

4. **Run Development Server:**
   ```bash
   npm run dev
   ```
